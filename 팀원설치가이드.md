# 카드뉴스 자동화 설치 가이드

## 사전 설치 (최초 1회)

### 1. Node.js
https://nodejs.org → **LTS 버전** 다운로드 후 설치

설치 확인:
```
node -v
```
`v18` 이상이면 OK


### 2. 소스코드 설치

터미널(cmd)에서 순서대로 실행:

```
git clone https://github.com/gobangMkt/insta-auto-cardnews
cd insta-auto-cardnews
npm install
```

---

## 사용법

### 1. ZIP 파일을 `대기중/` 폴더에 넣기
```
대기중/7866.zip
```
> 파일명 = 콘텐츠 번호. 포맷 모르면 담당자한테 확인.

### 2. `처리.bat` 더블클릭 → Claude 창 열리면 `로컬 처리` 입력

Claude가 자동으로:
- URL 크롤링
- 슬라이드 HTML 생성
- PNG 변환

### 3. `완료/` 폴더에서 결과 확인
```
완료/7866-result.zip  ← 압축 풀면 PNG 파일들
```

## 최신 버전 받기 (가끔)

```
cd insta-auto-cardnews
git pull
```

---

## 문제 해결

| 증상 | 해결 |
|------|------|
| `node -v` 안 됨 | Node.js 재설치 |
| `claude --version` 안 됨 | Claude Code 재설치 |
| `npm install` 오류 | Node.js 버전 확인 (v18 이상) |
| 처리.bat 실행 후 아무것도 안 됨 | `대기중/` 폴더에 ZIP 있는지 확인 |
| PNG 안 나옴 | 담당자에게 문의 |
